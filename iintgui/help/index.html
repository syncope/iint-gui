iint-gui

iint-gui is designed to facilitate direct and interactive analysis of data collected at P09.

The program is composed of several GUI Elements , plus there are several Menu Items available.

To use the program, several subsequent steps are performed.
  1. Data reading
  2. Definitions for the intensity
  3. Definitions for background
  4. Signal fitting
  5. Analysis and finalization
  6. Analysis and finalization
Every processing can be described by a configuration, which can be saved in configuration files
